# NEXA - Sitio Web Institucional

## Descripción
**NEXA** es un sitio web institucional desarrollado en HTML y CSS que presenta información sobre una empresa de desarrollo web enfocada en negocios locales de Coronel Brandsen y alrededores.

El objetivo del proyecto es mostrar de forma clara y moderna quiénes somos, nuestra historia, capacidades y objetivos, junto con una galería visual.

## Características
- Diseño moderno con gradientes y efecto glassmorphism  
- Navegación fija con scroll suave  
- Secciones organizadas y bien estructuradas  
- Galería de imágenes responsive  
- Efectos hover interactivos  
- Sitio adaptable a distintos dispositivos
